
### 항해 커뮤니티(Notice Board !)

- 항해에서 익명으로 서로 주고받고 싶은 이야기가 있을 때 사용할만한 커뮤니티 공간이 있으면 좋을 것 같다는 생각이 들어서
만들게 다음과 같은 프로젝트를 진행하게 되었습니다.

- 팀원
    + 서현웅(팀장) :: 메인페이지 , 댓글 호출 , 댓글 작성기능 , 게시글 호출
    + 박용원(팀원) :: 게시글 작성 페이지
    + 권용준(팀원) :: 게시글 호출 기능
    

- 주요기능
    + 게시글 작성
    + 해당 게시글에서 댓글 작성
    + 해당 댓글 삭제 그리고 게시글 삭제
    + 댓글들 수정 그리고 게시글 수정
